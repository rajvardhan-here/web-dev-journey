## 📖 About

I started learning web development, took a break along the way, and now I'm **restarting with full commitment**. The plan: master **frontend fundamentals** first (HTML → CSS → JS → React), while picking up **backend basics (Java)** in parallel — learning everything through an **API-first** approach rather than surface-level tutorials.

**Goal:** Practical full-stack mastery in **3 months** — real, deployable projects, not just concepts.
